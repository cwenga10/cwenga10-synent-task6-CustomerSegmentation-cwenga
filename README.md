
# Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project performs customer segmentation using machine learning (K-Means Clustering). The goal is to group mall customers based on their purchasing behavior using features like Annual Income and Spending Score.
---

## 📊 Dataset
Mall Customers Dataset  
Source: Kaggle (Mall Customer Segmentation Data)

Features used:
- Annual Income (k$)
- Spending Score (1–100)

---

## 🎯 Objective
To identify hidden customer groups and understand purchasing behavior for better marketing strategies.

---

## ⚙️ Workflow

### 1. Data Cleaning
- Checked missing values
- Verified data types
- Removed unnecessary columns

### 2. Exploratory Data Analysis (EDA)
- Distribution of income
- Spending score patterns
- Relationship between features

### 3. Model Building
- Applied K-Means Clustering
- Used Elbow Method to find optimal clusters
- Selected best K value

### 4. Visualization
- Scatter plot of customer segments
- Cluster labeling

---

## 📈 Results
The model successfully identified different customer groups such as:
- High income, high spenders
- Low income, high spenders
- Low spenders
- Moderate customers

---

## 🧠 Key Insights
- Customers can be grouped based on spending behavior
- Spending score is not always dependent on income
- Marketing strategies can be customized per segment
